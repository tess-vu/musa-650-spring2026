# Week 06 Test 2b — Introduction to Deep Learning

This folder contains materials for an introductory Deep Learning session, covering the core concepts of neural networks and applying them to image classification with the MNIST dataset.

## Contents

- [`week08_presentation.slides.html`](week08_presentation.slides.html) — Presentation
- [`week08_lab_student.ipynb`](week08_lab_student.ipynb) — hands-on MLP lab (student version with blanks)
- [`week08_lab_solutions.ipynb`](week08_lab_solutions.ipynb) — completed lab solutions

> [!NOTE]
> The lab materials can be run on Google Colab. You can open them there and execute the notebooks, or alternatively, run them locally by creating an environment with the required packages.
